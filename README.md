## Servidor UDP

Kevin Muñoz

La aplicación al ejecutarse abrirá la respectiva ventana de JavaFX donde se muestra la calculadora con la información respectiva para hacer las operaciones y enviar de forma correcta al servidor. Para estar pruebas se usa la dirección de loopback como valor por defecto para pruebas dentro de la misma máquina, pero permitiendo modificar de ser necesario para colocar una IP local que usa la red interna.
